# Helmet-Triple-Riders-Detection-AI
AI-based helmet and triple rider detection system

## Description
This project detects whether riders are wearing helmets and identifies triple riders on vehicles using AI-based computer vision. It aims to improve road safety and automate traffic monitoring.

## Features
- Detects helmet usage in real-time.
- Detects triple riders in vehicles.
- User-friendly web interface using Flask.
- Supports live camera or video input.
- Generates alerts for traffic violations (optional extension).

## Technologies Used
- Python
- YOLOv8 (Object Detection)
- OpenCV (Image & Video Processing)
- Flask (Web Frontend)
- SQLite3 (Database, if used)
